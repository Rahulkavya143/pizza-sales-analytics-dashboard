# Pizza Sales Analytics Dashboard

## Project Overview

Developed an end-to-end Pizza Sales Analytics Dashboard using SQL and Power BI to analyze sales performance, customer ordering trends, and product performance.

## Business Problem

The objective was to analyze pizza sales data and identify key business insights such as:

* Total Revenue
* Average Order Value
* Total Orders
* Best-Selling Products
* Worst-Selling Products
* Daily and Monthly Sales Trends
* Category and Size Performance

## Tools Used

* SQL
* Power BI
* Microsoft Excel

## SQL Concepts Used

* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY
* ORDER BY
* DISTINCT
* Subqueries
* KPI Calculations

## Dashboard Features

* Revenue KPI Tracking
* Daily Trend Analysis
* Monthly Trend Analysis
* Category Performance Analysis
* Best & Worst Seller Analysis
* Interactive Filters and Navigation

## Key Insights

* Generated $817K+ Revenue
* Analyzed 49K+ Pizza Sales Records
* Identified Top Performing Products
* Identified Low Performing Products
* Evaluated Category-wise Sales Contribution
